**Dead or Alive in action**


A simple tool to check whether a particular host is alive or not!

# Prerequisites
A *nix system
# Installation
```
chmod +x install.sh
./install.sh
```
Then from the next time just type in 
```
DOA
```
in the terminal to start the tool!!
